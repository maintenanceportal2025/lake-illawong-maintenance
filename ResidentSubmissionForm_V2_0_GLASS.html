❓ How to Submit a Problem Report
⚠ CRITICAL: You MUST select from the dropdown lists — do not type the full value
1️⃣ Enter Location
  • Type your unit number (e.g. 12) or area name
  • Wait for the dropdown list to appear
  • Click your location from the list
  • ❌ Do NOT type the full name and press Enter
2️⃣ Enter Reported By
  • Type the first 2-3 letters of your name (e.g. jo)
  • Wait for the dropdown list to appear
  • Click your name from the list
  • ❌ Do NOT type your full name and press Enter
3️⃣ Describe the Problem
  • Type a clear description of the problem
  • The more detail the better
💡 Why must I use the dropdowns? The system needs exact matches to assign your problem to the correct Zone Representative and send notifications to the right people. Typing free text creates errors that result in missing information and delays.
